# 启动步骤

1. 把[SQL](/SQL)里的文件导入MySQL
2. [Back-end](/Back-end)为Spring Boot项目
3. [Mid-end](/Mid-end)为Ant Design的管理端
    ```cmd
    npm install
        or
    yarn
    ```
4. [Front-end](/Front-end)为React项目
    ```cmd
    npm install
        or
    yarn
    ```
